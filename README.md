## 📖 About this software
This tool helps researchers and students find water ice under the surface of the Moon. It focuses on the Faustini crater at the lunar south pole. The software uses data from the Chandrayaan-2 satellite to map these hidden resources. 

The pipeline combines radar data with high-resolution images. It uses math models to predict where ice exists. A machine learning system checks these results to ensure accuracy. This process helps you understand the moon's geography without needing deep technical knowledge of space physics.

## ⚙️ Using the Interface
The application dashboard divides the work into three steps. You can select your desired region of interest using the interactive map.

### Step 1: Select Your Region
Click on the map of the Faustini crater. You can use your mouse to zoom into specific sections. The application highlights areas where data exists. When you click an area, the program loads the radar data for that coordinate.

### Step 2: Run the Analysis
Press the Start Analysis button. The software uses a model to scan the selected area. This process looks for signals that indicate frozen water. The tool displays a progress percentage as it finishes the calculations.

### Step 3: View the Results
The results appear as a color-coded map. Blue areas indicate a high probability of water ice. You can click these areas to see charts that explain the findings. The software provides a summary table that lists the estimated depth and volume of the ice. You can save these reports to your computer as text files for later review.

## 📋 Frequently Asked Questions

**Does this software require a paid license?**
No. This tool is free for everyone to use and distributed through this platform.

**Can I use this tool for regions outside the south pole?**
The current version focuses only on the Faustini region to maintain high accuracy. Future updates may include other lunar craters.

**How does the system know where ice exists?**
The software examines radar reflections from the lunar surface. Water ice changes the way radar waves bounce back to the satellite. The machine learning model identifies these specific patterns and matches them against known geological data.

**Do I need a powerful graphics card?**
No. The software performs most tasks using your main processor. It functions well on standard laptops and tower computers.

**How do I clear the cached data?**
Go to the settings menu inside the application. Click the button labeled Clear Cache. This deletes temporary files and frees up space on your hard drive.
